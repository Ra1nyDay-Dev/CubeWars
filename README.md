<H2>Description</H2>
Quake-like isometric 3D shooter with a classic single-weapon deathmatch mode.

This is an early prototype build. The game is in active development, so many features are still rough and may contain bugs.

The store, character, and settings are not yet available.
<BR>

<H2>Play game</H2>
<UL>
<li>GitHub pages: https://ra1nyday-dev.github.io/CubeWars_Build_Web/</li>
<li>Itch.io: https://ra1nydays.itch.io/cubewars</li>
</UL>

<H2>Controls</H2>
<UL>
<li>Move: WASD</li>
<li>Jump: Space</li>
<li>Interact/Take weapon: E</li>
<li>Reload: R</li>
<li>Attack and Aim: Mouse buttons</li>
</UL>
The game also supports gamepad, but only during a match.
